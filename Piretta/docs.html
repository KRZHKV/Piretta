Создание документации по обращению с ОМО для ЛПУ

Составление схемы сбора и перемещения ОМО в ЛПУ, инструкций для персонала и их модификации при изменении загрузки внутри ЛПУ.

В рамках комплексного предложения по организации обращения, хранения и утилизации опасных медицинских отходов наша компания обеспечит:

  • Обследование медучреждения (аудит по медотходам), определение точного количества необходимых упаковочных материалов (пакетов и иглосборников, согласноСанПиН 2.1.7.2790-10)
  • Создание схем движения ОМО внутри и вне медицинского учреждения
  • Оборудование и распределение грязных/чистых зон
  • Подготовка памяток и обучающих материалов для медицинского персонала
  • Создание соответствующего документооборота внутри и вне медицинского учреждения
  • Установка информационных стендов
